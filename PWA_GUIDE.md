# 积分兑换 App - PWA 使用说明

## 什么是 PWA？

PWA（渐进式 Web 应用）是一种让 Web 应用可以像原生 App 一样在手机上运行的技术。

## 如何在手机上使用？

### Android 手机
1. 打开 Chrome 或 Edge 浏览器
2. 访问应用网址
3. 点击浏览器右上角的菜单（三个点）
4. 选择「添加到主屏幕」或「安装应用」
5. 完成后，你会在手机桌面上看到应用图标，可以像普通 App 一样打开使用

### iPhone (iOS)
1. 打开 Safari 浏览器
2. 访问应用网址
3. 点击底部的分享按钮（方形带向上箭头的图标）
4. 向下滑动，选择「添加到主屏幕」
5. 点击「添加」，应用就会出现在你的 iPhone 桌面上

## 优势

✅ **无需安装 APK/IPA** - 直接在浏览器中访问即可添加
✅ **跨平台** - Android 和 iPhone 都可以使用
✅ **离线可用** - 可以在没有网络的情况下打开应用
✅ **快速启动** - 像原生 App 一样启动速度快
✅ **自动更新** - 每次打开都会自动获取最新版本

## 本地测试

```bash
# 启动开发服务器
pnpm run dev

# 或者构建并预览生产版本
pnpm run build
pnpm run preview
```
